# SalesWebMVC

[![CI - SalesWeb](https://github.com/ViniMartins10/SalesWeb/actions/workflows/ci.yml/badge.svg)](https://github.com/ViniMartins10/SalesWeb/actions/workflows/ci.yml)

Projeto ASP.NET Core MVC criado para praticar desenvolvimento web com C#, Entity Framework Core e banco de dados relacional.

## CI/CD

Este projeto possui uma pipeline de integracao continua com GitHub Actions que executa automaticamente:

- restauracao de dependencias;
- compilacao em modo Release;
- execucao de testes.

A pipeline roda a cada push ou pull request para a branch `main`.
